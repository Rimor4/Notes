# Markdown快捷键

## 一、核心文本格式快捷键

| 功能         | 语法                   | 快捷键              |
| ------------ | ---------------------- | ------------------- |
| 字体加粗     | `**加粗文本**`         | Ctrl + B            |
| 斜体字       | `*斜体文本*`           | Ctrl + I            |
| 加粗斜体     | `***加粗斜体文本***`   | Ctrl + B + Ctrl + I |
| 删除线       | `~~删除线文本~~`       | Alt + Shift + 5     |
| 下划线       | `<u>下划线文本</u>`    | Ctrl + U            |
| 短代码块     | ``短代码内容``         | Ctrl + Shift + `    |
| 超链接       | `[链接文本](链接地址)` | Ctrl + K            |
| 插入图片     | `[图片名称](图片地址)` | Ctrl + Shift + I    |
| 清除文本样式 | -                      | Ctrl + \            |

## 六、特殊操作快捷键

| 功能                 | 操作方式                     | 快捷键 / 语法      |
| -------------------- | ---------------------------- | ------------------ |
| 实星标记             | 星号 + 空格                  | `* `（符号后空格） |
| 空心标记             | 空格 + 星号 + 空格           | `*`（前后空格）    |
| 分割线               | 3 个连续星号 / 减号 / 下划线 | `***`/`---`/`___`  |
| 计划任务（未完成）   | - [ ] + 任务名               | `- [ ] 未完成任务` |
| 计划任务（已完成）   | - [x] + 任务名               | `- [x] 已完成任务` |
| 首行缩进 1 个汉字    | 使用特殊符号                 | ` `                |
| 首行缩进 0.5 个汉字  | 使用特殊符号                 | ` `                |
| 首行缩进 0.25 个汉字 | 使用特殊符号                 |                    |
